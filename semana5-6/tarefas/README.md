# Gerenciadores de Tarefas em Python

Esta pasta contém três versões de **gerenciadores de tarefas** desenvolvidos em Python:

1. **CLI completa com status e prazo** – armazena tarefas com título, prazo e status de conclusão (`tarefas_cli.py`).  
2. **GUI com PySimpleGUI** – interface gráfica para adicionar, remover e concluir tarefas (`tarefas_gui.py`).  
3. **CLI simples** – versão minimalista, apenas strings de tarefas (`tarefas_simples.py`).  

---

## Funcionalidades

### 1. CLI completa (`tarefas_cli.py`)
- Adicionar tarefas com título e prazo (YYYY-MM-DD)  
- Marcar tarefas como concluídas  
- Remover tarefas  
- Listar tarefas com status (`✔️` ou `❌`) e prazo  
- Persistência via JSON (`tarefas_cli.json`)  

### 2. GUI com PySimpleGUI (`tarefas_gui.py`)
- Adicionar tarefas  
- Concluir tarefas  
- Remover tarefas  
- Listar tarefas em uma interface gráfica  
- Persistência via JSON (`tarefas_gui.json`)  

### 3. CLI simples (`tarefas_simples.py`)
- Adicionar tarefas como texto  
- Remover tarefas  
- Listar tarefas  
- Persistência via JSON (`tarefas.json`)
