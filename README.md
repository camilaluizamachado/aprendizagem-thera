## Introdução  
Este repositório foi criado para ser o meu guia central dentro do programa — tanto para mim, como nova integrante, quanto para os monitores e coordenadores que me acompanham.  

## Objetivo  
Meu objetivo aqui é me capacitar a partir de uma base sólida, evoluindo até me tornar uma profissional capaz de colaborar de forma efetiva em projetos reais de desenvolvimento de software. Quero dominar tanto as **competências técnicas (hard skills)** quanto as **comportamentais (soft skills)** que são essenciais no ambiente de trabalho.  

## Pilares do Programa  
- **Autoaprendizado Guiado:** o estudo e a prática são minha responsabilidade. O monitor atua como meu guia e revisor.  
- **Aprendizado Baseado em Projetos:** busco construir conhecimento por meio da prática e da resolução de problemas reais.  
- **Comunicação Clara:** tudo que eu fizer será bem documentado e discutido de forma aberta e transparente.
