# Sistema Integrado de Agenda e Tarefas em Python

Este projeto contém um **sistema integrado** para gerenciar **contatos** e **tarefas** utilizando Python e arquivos JSON para persistência de dados.  

Ele possui uma estrutura modular, separando agenda e tarefas, permitindo fácil manutenção e expansão.

---

## Funcionalidades

### Agenda de Contatos
- Adicionar contatos (nome e telefone)  
- Listar contatos salvos  
- Persistência via `agenda.json`  

### Gerenciador de Tarefas
- Adicionar tarefas (apenas nome da tarefa)  
- Listar tarefas  
- Remover tarefas  
- Persistência via `tarefas.json`  

### Sistema Integrado
- Menu principal para escolher entre Agenda e Tarefas  
- Retorno ao menu principal após concluir operações  

---


## Como Rodar

### 1. Clone o repositório e entre na pasta:


git clone <URL_DO_REPOSITORIO>
cd projeto_integrado
Execute o sistema integrado:


python main.py
Use os menus:

1 → Agenda
2 → Tarefas
3 → Sair
